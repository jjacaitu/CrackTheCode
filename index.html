Crack The Code|

  • A game inspired by mastermind.
  • You will have to guess a 4 digit code in less than 10 attempts or before times runs out in order to stop the launch of a nuke. Don't forget to press the stop button when you crack the code.
  • Everytime you enter the wrong code the timer will go down by 2 minutes.
  • Each time you enter a 4 digit code, your input will be validated and the computer will give you feedback.
  • Feedback will be given by 4 color boxes. A green box will be shown for each digit of the code that is correct on both number or color and position. A red box will be shown for each digit that exists in the code but its not in the correct position.
  • The colored boxes doesn't represent the digit position.
  • Numbers can repeat on the code.
  • Use this feedback to try to make a better guess in your next turn.
